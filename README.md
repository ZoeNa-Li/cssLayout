##text-align: center 文本水平居中
##display:table; 文本/盒子 水平居中
##指定宽度  margin: 0 auto;盒子水平居中
##position: absolute;水平居中
##flex+justify-content: center;水平居中
##父flex + 子margin: 0 auto;水平居中
